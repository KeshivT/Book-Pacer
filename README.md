# Book-Pacer
Book Pacer Website
Designed and created by Keshiv Tandon
Features:
1. Users can create accounts where all of their books are stored
2. Allows for book uploads of most formats
3. Converts uploaded books to text, displaying them chapter-by-chapter on website.
4. Chapters are automatically locked, then unlock at a random time (range set by user per book)
5. Email sent to user upon chapter unlocking
